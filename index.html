Resultados

Qué serás capaz de hacer

Al finalizar el programa podrás construir soluciones de datos completas, de forma autónoma, que impacten decisiones reales en tu organización.

🔌

Conectar y transformar datos

Excel, CSV, SharePoint, SQL Server y APIs web. Limpiar y preparar datos con Power Query sin necesidad de código.

🗂️

Construir modelos dimensionales

Diseñar Star Schemas con tablas de hechos y dimensiones. Relaciones, cardinalidad y contexto de filtro.

⚡

Escribir medidas en DAX

Desde SUM hasta Time Intelligence. YTD, YoY, Rankings, KPIs dinámicos y medidas con CALCULATE.

📊

Diseñar visualizaciones efectivas

Seleccionar el visual correcto para cada pregunta. Evitar los errores más comunes. Drillthrough y tooltips.

📖

Contar historias con datos

Aplicar los principios de Cole Nussbaumer Knaflic. Reducir ruido visual, guiar la atención y comunicar decisiones.

🏢

Publicar con seguridad

Power BI Service, Workspaces, Apps y Row Level Security. Gobierno básico y actualizaciones automáticas.

Para quién es

Requisitos y audiencia

✅ Objetivos al finalizar

  • ✓ Crear reportes y dashboards profesionales de forma autónoma
  • ✓ Conectar, limpiar y transformar datos de múltiples fuentes
  • ✓ Diseñar modelos dimensionales correctos
  • ✓ Escribir medidas DAX incluyendo Time Intelligence
  • ✓ Aplicar principios de Storytelling con datos
  • ✓ Publicar y gestionar reportes en Power BI Service
  • ✓ Presentar hallazgos a audiencia ejecutiva

📋 Requisitos previos

  • Conocimientos básicos de Excel (tablas, fórmulas simples)
  • Computadora con Windows 10/11
  • Power BI Desktop instalado (gratuito)
  • Cuenta Microsoft 365 (para Power BI Service)
  • Disposición para practicar entre sesiones
  • No se requieren conocimientos de programación

👥 Audiencia

  • Usuarios de negocio y analistas junior
  • Personal administrativo y financiero
  • Coordinadores de área y project managers
  • Profesionales de ONG y fundraising
